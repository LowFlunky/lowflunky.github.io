Skill Sets

An overview of my Hard and Soft skills gained through Employment prior to University, as well as during my time studying.

Due to skills being hard to judge in terms of level of work, I thought a simpler solution was to list them with various confidence level meters to each. Look below for each, with a bit of description and confidence rating.

Programming/Coding

C#

This has been the primary focus during the first stage of University. Coding within Unity, Visual Studio and Netbeans to create a wide array of programs, procedural, object orientated and event driven, creating solutions for various problems or, more importantly, to create/break/fix our own Game projects.

95% Confident, you can never be 100%

Java

Only starting Java recently, hands on experience is limited yet the sheer similarity to C# makes it a breeze to use. A few searches for syntax differences and structural changes in the implementation of our software means we can continue utilising knowledge from the previous year whilst experiencing a new language.

80%

C++/C

During the 2 years of College, previous to University, we predominantly programmed in C++, however due to the nature of College and the time since leaving I have not returned to C++ and C development. That being said my understanding of the theory and coding paradigms that they implement has increased greatly within the last few years. If using these languages again a quick refresher would be in order but there wouldn't be any drama in that.

65%

Web Development/Design

HTML5

I find the implementation of HTML5 intuative and easy to pickup and put-down whenever I need. The new features 5 brings really opens the way for some amazing web pages, without the need for fancy web players or plugins. An exciting time to be a web developer I'm sure.

90% Confident

CSS (Cascading Style Sheets)

CSS is an easy and flexible way to manage multiple pages styles via 1, or more, documents, as the name says the changes cascade through all pages linked to it. However I am not a designer at heart, in the sense of artistic flare and style, so I find that this is a powerful tool that I would like to improve my skills with.

70%

JavaScript

Originally I found JavaScript awkward and limited, more through my own lack of understanding, however after making a game in JavaScript in year 1 of University I was blown away with the posabilities this holds. My knowledge of this is average due to the lack of time spent really getting into the meat of it, especially with powerful plug-ins such as Phaser

50%

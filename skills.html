Web Development/Design/Communication

HTML5

I find the implementation of HTML5 intuitive and easy to pick up and put-down whenever I need. The new features that 5 brings really opens the way for some amazing web pages, without the need for fancy web players or plugins. An exciting time to be a web developer I'm sure.

90% Confident

CSS (Cascading Style Sheets)

CSS is an easy and flexible way to manage multiple pages styles via 1, or more, documents, as the name says the changes cascade through all pages linked to it. However I am not a designer at heart, in the sense of artistic flare and style, so I find that this is a powerful tool that I would like to improve my skills with.

70%

JavaScript

Originally I found JavaScript awkward and limited, more through my own lack of understanding, however after making a game in JavaScript in year 1 of University I was blown away with the possibilities this holds. My knowledge of this is average due to the lack of time spent really getting into the meat of it, especially with powerful plug-ins such as Phaser.

50%

PHP

Using PHP predominantly in College, I find that it was simple to use once you have the systems in place to run it. The hardest part appeared to be starting, getting XAMP in a working state and working out the htdocs is you localhost folder. Once created PHP is useful server side tool that really adds to HTML's features, logins, registrations, cookie handing it’s all simple, however the syntax is a tricky beast that can really throw a fit due to the ease of error rate, a brace in the wrong place and you’re going to spend a lot of time working out your mistakes.

65%

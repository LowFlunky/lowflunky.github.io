Software Projects

The following pieces of software have been created using C# in either Visual Studio, NetBeans or with HTML5, CSS and Javascript in Sublime Text.

MasterMind

Based on the board game, the player must guess a pattern of "colors", represented as numbers that match the computers generated pattern. Black response mean correct number but wrong position, White mean correct number in the correct place. This was to test our understanding of variable handling and manipulation, alongside Algorithms that check against random generated arrays of data, to find patterns and allow a win state. This also includes a structogram design document.

Requires Visual Studio 2010/2012

Conversion

A group based project where we had a .txt file with conversion rules, the software would read this file then, using it as a template, allow users to convert between varying types with appropriate exception handling to make sure it was valid. This introduced the concepts of validation alongside Stream readers, as well as manipulation of user input to circumvent errors within upper case and lower case mismatches.

Single .cs file, compile in Visual Studio 2010/2012.

Website

This project was to take a poorly made website, I chose a local school, and edit its layout and style to meet theoretical goals. These goals would assist in user navigation, lower the amount of clicks it requires to find information and streamline the website so that first time users could use already known mental patterns to navigate, such as static navigation.

Code requires any text editing tool, notepad++ for example, and can be run in any browser.
